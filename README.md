# Contacts-app

Contacts-app — Android-приложение для просмотра контактов устройства и удаления повторяющихся записей контактов.

## Стек

- **Языки:** Java, Kotlin.
- **UI:** Jetpack Compose, Material 3.
- **IPC:** Android Service + AIDL.
- **Сборка:** Gradle, Android Gradle Plugin.
- **Тестирование:** JUnit unit-тесты.

## Возможности

- Просмотр контактов устройства.
- Группировка контактов по буквенным секциям.
- Автоматическое обновление списка при изменении телефонной книги.
- Поиск повторяющихся контактов.
- Выполнение операций с контактами через отдельный Android Service.
- Взаимодействие с сервисом через AIDL callbacks.
- Отображение результата операции:
    - дубли удалены;
    - дубли не найдены;
    - произошла ошибка.

## Требования

- Android Studio.
- Android SDK API 36.
- JDK 21 для Gradle.
- Android-устройство или эмулятор с поддержкой контактов.
